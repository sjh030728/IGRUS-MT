# 의존성 주입 (dependency injection)

**한 줄:** 필요한 걸 안에서 직접 만들지(import하지) 않고 밖에서 건네받는 것.
받는 쪽은 "뭘 주든 이 모양이면 된다"만 알아서, 진짜 대신 가짜([[스텁]])도 끼울 수 있다.

어디서 나왔나: `quiz.game.ts`의 `makeQuizGame(loadQuestions)` — 게임이 DB를 직접
잡지 않고 적재 함수만 주입받는다. Nest의 생성자 주입(`constructor(private db: DbService)`)도
같은 개념의 프레임워크판.
