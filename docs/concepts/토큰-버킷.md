# 토큰 버킷 (token bucket)

**한 줄:** 요청을 처리할 때마다 양동이에서 토큰을 꺼내 쓰고, 토큰은 일정 속도로만 다시
차오르게 하는 속도 제한법. **평균은 묶되 순간 폭발은 허용**하는 게 고정 상한과의 차이다.

어디서 나왔나: `live.service.ts`의 `tap()` — 탭 줄다리기 레이트 상한. 용량 25(burst) /
충전 15개/초(perSec). 진짜 잘 치는 사람의 순간 연타(20+/s)는 통과시키고, 그걸 25초 유지하는
스크립트만 걸러낸다 — "진짜 잘 치는 사람이 치터로 잡히는 게 훨씬 나쁘다" (anticheat.ts).
