# 순수 함수와 결정성

> **스텁.** `/learn 순수 함수` 로 물어보면 채워진다. #스텁/성장

**한 줄:** 같은 입력에 항상 같은 출력을 내고 바깥 세상을 안 건드리는 함수.

**이 프로젝트에서:**
- `packages/protocol/src/game.ts` — `score()`와 `parseAnswer()`에 "순수·결정적"이 계약으로
  박혀 있다. `loadRounds()`엔 "반드시 결정적일 것. ORDER BY random() 금지"

**왜 계약에 박았나:** 되감기와 재시작 복구가 "다시 돌려도 같은 답이 나온다"에 의존한다.
`score()`가 현재 시각이나 난수를 보면 무효 후 재실행에서 다른 점수가 나온다.

**궁금해할 만한 것:** `loadRounds`가 `async`인데 순수할 수 있나? "결정적"과 "순수"가
같은 말인가? (힌트: DB를 읽는 건 부수효과인가?)
