# 굽기 (bake in)

**한 줄:** 나중에 정해도 될 걸 **지금 구조에 박아서** 나중에 못 바꾸게 만드는 것.
반죽은 되돌릴 수 있지만 **구운 다음엔 계란을 못 꺼낸다.**

## 왜 필요한가 — 트리는 순서를 하나만 표현한다

노트를 폴더로 정리한다고 치자: `concepts/실시간/스텁/…`. "실시간 전부"는 한눈에 보이는데
"스텁 전부"는 폴더를 다 열어봐야 한다. 순서를 뒤집으면 반대쪽이 흩어진다.
**트리에서 위로 올린 축은 보이고, 밑으로 내린 축은 흩어진다** — 파일은 폴더 하나에만 있을 수
있어서 피할 방법이 없다. 그게 굽는 것이다: 두 사실의 관계를 경로 문자열에 박았다.

서로 아무것도 안 말해주는 두 사실(**직교, orthogonal**)을 트리에 넣으면 없는 관계를
지어내게 된다("당일은 실시간의 한 종류" — 거짓이다. 그냥 둘 다 참일 뿐).
그래서 태그를 나란히 두 개 달고 교집합은 검색이 한다: `tag:#영역/실시간 tag:#스텁`.
**트리에 굽지 않아도 질의는 되고, 구우면 못 되돌린다** — 되돌리는 비용이 파일 개수에
비례하고, 안 굽는 비용은 0이다.

Docker 이미지에 "굽는다", 게임 라이팅을 "굽는다", "baked-in 가정" — 전부 같은 뜻이다:
뒤 단계에서 정할 수 있는 걸 앞 단계에 고정한다.

## 이 프로젝트에서

**굽는 게 맞았던 것 — `postgres:16`.** docker-compose에 버전을 숫자로 박는다.
`latest`(안 굽기)를 쓰면 다음 노트북에서 다른 버전이 받아진다 — **재현하려고 쓰는 도구가
재현을 깬다.** 재현성을 살 땐 굽는 게 맞다 → [[docker와-컨테이너]]

**값은 안 굽고 구조만 구운 것 — `RateCap`** (`anticheat.ts`). 계약 스키마는 "상한이
존재한다"(구조)만 못 박고 값은 안 박았다 — 진짜 값은 리허설 실측에서 나오기 때문이다.
**답을 줄 사건이 예정돼 있으면 그때까지 안 굽는다.** (단계 3에서 시작값이 잠깐 같은 파일에
들어왔다가 게임 모듈 상수로 옮겨졌다 — `tap-tug.game.ts`의 `RATE`. K·DURATION과 같은
"리허설이 정하는 자리"고, 실측 반영이 계약 수정이 되지 않는다. config로 안 뺀 이유:
치팅 방어의 상한은 7:30에 사람 손이 닿으면 안 되는 값이다. 제정신 검사(burst > perSec)는
부팅 프리플라이트가 한다.)

**굽되 이유를 함께 박은 것 — `MAX_CONTENT_CHUNKS = 3`** (`display.ts`). 설정으로 빼면
유연해지는 게 아니라 아무도 몇이 맞는지 모르게 된다. 사람 판단에서 나온 숫자는
상수 + 이유 주석이 답이다 → [[하드코딩]]

## 하드코딩과 뭐가 다른가

| | 뭘 잘못하나 | 어디가 아픈가 |
|---|---|---|
| [[하드코딩]] | 같은 진실이 **두 곳에** 산다 | **발견이** 어렵다 — 조용히 어긋난다 |
| 굽기 | 나중에 정해도 될 걸 **지금** 정한다 | **고치기가** 어렵다 — 전부 건드려야 한다 |

하드코딩은 공간의 문제(몇 벌 있나), 굽기는 시간의 문제(언제 정하나).

## 언제 굽고 언제 안 굽나

- **굽는다:** 재현성을 살 때(`postgres:16`) / 결정을 기록할 때(상수 + 이유) / 정할 주체가 없을 때
- **안 굽는다:** 두 축이 직교할 때(태그로) / 답을 줄 사건이 예정돼 있을 때(`RateCap`) /
  질의로 될 때(검색이 답을 주는데 트리를 만들면 이득은 겹치고 비용만 는다)
- **판단 비대칭:** 안 구우면 조합 전부 + 되돌림 비용 0, 구우면 한 방향 브라우징.
  **둘 다 답이 나오면 되돌릴 수 있는 쪽을 고른다.**
- **★남용 지점 — "굽지 마라"가 답이 아니다.★** 굽기의 반대는 "안 정하기"가 아니라
  "다른 데서 정하기"다(`RateCap`은 리허설이, `POSTGRES_PASSWORD`는 환경이 정한다).
  정할 주체 없이 안 구우면 그냥 안 정해진 것이다 — 설정 항목 100개짜리 앱이 그렇게 나온다.
  유연성을 산 게 아니라 결정을 안 한 것이다.
